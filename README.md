Project Overview:

This repository contains my Digital Literacy Project completed as part of the CSE0001 course at VIT Bhopal. The project focuses on building essential digital skills such as creating a professional online presence, communicating effectively, understanding safe internet practices, and spreading awareness about cybercrime.

The project is divided into five tasks, each aligned with a module of the course, covering infographic creation, digital portfolio development, platform exploration, email etiquette, and cybercrime awareness.

Module Summaries:

 Task 1 – Digital Literacy Infographic
Created an infographic using Canva explaining digital literacy concepts, tools, and safe practices.
Focused on awareness and easy understanding for students.
 Task 2 – Student Digital Portfolio
Set up profiles on platforms like GitHub, LinkedIn, and Kaggle.
Built a basic professional online presence for future opportunities.
 Task 3 – Coding & Collaboration Platforms
Completed a beginner coding challenge on (HackerRank/CodeChef).
Created a Google Form quiz for digital literacy awareness.
 Task 4 – Email Etiquette & Communication
Wrote two professional emails:
Request for assignment extension
Internship application
Created a social media do’s and don’ts checklist.
 Task 5 – Cybercrime Awareness
Developed a case study on (Phishing / UPI Fraud / etc.).
Created a prevention checklist for students.
