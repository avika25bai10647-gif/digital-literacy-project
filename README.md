Project Overview

This repository contains my Digital Literacy Project completed as part of the CSE0001 course at VIT Bhopal.
The project focuses on building essential digital skills such as online presence, professional communication, safe internet practices, and awareness of cybercrime.

It includes five tasks covering infographic design, digital portfolio creation, platform exploration, email etiquette, and cybercrime awareness.
